Walkthrough

Here you have a basic and quick example of a walkthrough

{% include code/walkthrough.html %}

Step

First of all, each step belongs to a category (and it can only have one) and it has an 'options' param. Those are the basic params that a step should have.

For each category the options are different, here you have the list of categories. We are going to explain in detail each one. Even if the step has multiple options, for each step you can only set one of them

  • Layers
  • Tab
  • Timeline
  • Country
  • Analysis
  • Map

Layers

We need to set a 'layers' param equal to an array of layer slugs. You can find here the slugs Layerspec. If you want to set a country layer, you should first set the country

{% include code/layers.html %}

Tab

We can set an active tab or a subtab (like the analysis ones).

  • Country => 'countries-tab'
  • Analysis => 'analysis-tab'
  • Basemaps => 'basemaps-tab'
  • High res => 'hd-tab'
{% include code/tab.html %}

Timeline

We need to set a 'dates' param equal to an object with a 'begin' and 'end' param. Of course, the dates should be between the range of dates allowed by the current forest change layer. The format of the dates should be 'dd/mm/YYYY'

{% include code/timeline.html %}

Country

We need to set an 'iso' param with 2 params: 'country' and 'region'

{% include code/country.html %}

Analysis

We must specify the type of the analysis, so the 'analysis' param is mandatory. we have 3 different types of analysis:

  • 'draw' - Draw or upload shape
  • 'country' - Country or region
  • 'area' - Area
{% include code/analysis.html %}

Map

We have different possibilities here. We can set a zoom level, a location or a basemap. Just remember that you can only change one thing per step. For the basemaps, here you have a list of the ids of the basemaps.

  • Default => 'grayscale'
  • Terrain => 'terrain'
  • Satellite => 'satellite'
  • Dark Matter => 'dark'
  • Positron => 'positron'
  • Roads => 'hybrid'
  • Tree height => 'treeheight'
{% include code/map.html %}
